# 🎓 Student Result Management System

A web-based **Student Result Management System** developed using **HTML, CSS, and JavaScript**. The application enables administrators to manage student records, generate examination results, view saved results, and monitor overall statistics through an interactive dashboard.

## 🚀 How to Run the Project

1. Download or clone the repository.

```
git clone https://rsdec12.github.io/upskillCampus/
```

2. Open the project folder.

3. Run **index.html** in any modern web browser.

No additional software or database is required.

## 📊 Modules

### 1. Home
Provides an introduction to the Student Result Management System.

### 2. Login
Allows secure administrator access.

### 3. Dashboard
Displays:
- Total Students
- Results Generated
- Passed Students
- Failed Students

### 4. Student Management
- Add Student
- Edit Student
- Delete Student
- Search Student

### 5. Result Management
- Enter Marks
- Calculate Percentage
- Generate Grade
- Save Results

### 6. View Results
Displays all saved student results with search functionality.

### 7. About
Project information and developer details.

### 8. Contact
Contact form for communication.
