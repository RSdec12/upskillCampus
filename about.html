About the Project

The Student Result Management System is a web-based application developed using HTML, CSS and JavaScript. It helps educational institutions manage student records, calculate examination results and generate mark sheets efficiently.

💻 Technology Used

  • HTML5
  • CSS3
  • JavaScript (ES6)
  • Local Storage

✨ Project Features

  • Admin Login
  • Student Management
  • Result Generation
  • View Results
  • Print Result
  • Download PDF
  • Responsive Design
